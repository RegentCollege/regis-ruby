# REGIS

Used to access a REGIS api

## Installation

Add this line to your application's Gemfile:

    gem 'regis'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install regis

## Configuration

Set config/regis_api.yml

Set features/config/test_values.yml

## Usage

See features/step_definitions/api_steps
